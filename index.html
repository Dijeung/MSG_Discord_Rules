해당 규칙은 MSG 서버에서 작성되었으며,
내용을 허가 없이 복사 또는 수정하는 경우 처벌당할 수 있습니다.



제 1조 #MSG 서버 디스코드 기본규정

1. 모든 신고는 총관리자가 판단 하에 처벌이 결정됩니다.
2. 모든 유저는 규칙 위반자를 신고할 의무가 있으며 묵인, 방관 할 시 처벌당할 수 있습니다.
3. 서버 내 모든 채팅은 저장되며 관리진은 채팅을 볼 수 있으며 증거자료가 필요할 때 참고될 수 있습니다.
   ( 이 외의 목적으로 사용하지 않습니다 )

! 본 법전 게시글은 2021년 12월 10일부터 적용되며,
본 게시글에 기재되지 않은 사항이라도 서버대표 판단하에 처벌당할 수 있습니다.

! 처벌 방식 또한 관리자 판단하에 바뀔 수 있으며, 처벌 방식은 다음과 같습니다.
( 추방 / 밴 )

! 반성문을 통해 밴이 풀리게될때는 밴 사유에 따라 다음과 같은 처벌이 이루어집니다.
( 마이크 음소거 / 헤드셋 음소거 )

   - 밴 경력이 있을경우 경고 1회만 받더라도 밴처리 됩니다. 



먼저! 디스코드 서버에 들어오셨다면 별명 변경으로 이름 / 마크닉네임 으로 바꿔주시기 바랍니다.

제 2조 #채팅창 및 통화방

1. 상대방이 싫어하는 행위를 지속적으로 하는 행동
   (욕설이 아니여도 싸움을 일으키는 내용)

2. 공개된 자유 채팅방, 또는 통화방에서 싸우는 행동

3. 자유 채팅에 글, 노래, 사진, 홍보 등

4. 새로 들어온 뉴비 차별

5. 채팅, 통화방 독점

6. 타 서버를 하는 유저가 통화방에서 언급 & 방송하는 행위



제 3조 #사진 공유 채널

1. 자기 사진 이외 다른 사람 사진

2. 호불호 심하게 갈리는 취향 사진 (음식, 인물, 동물 등)

3. 어린이 친구들이 보면 안되는 사진 (19+)



제 4조 #마을 등록 채널

마을 디스코드방이 따로 있다면, 링크를 #마을 등록 채널에 올려주셔야 됩니다.
마을원이 아닌 사람이 해당 채널로 들어가서 민폐를 끼칠경우 신고가 가능합니다.

( 링크는 영구적으로 만들어주셔야 하며, 본 규칙을 위반하실경우 마을원 전부 처벌대상 입니다 )
( 관리진들이 감시를 위해 들어갈 수 있으니 이 부분 참고해주시기 바랍니다 )

  - 개인 메시지로 욕설 / 협박들은 신고가 가능합니다.



제 5조 #노래방 통화방

1. 사용중인 노래방 봇을 마음데로 조작할 경우 디스코드 밴처리 됩니다.

2. 노래봇을 사용하여 이상한 브금 / 19+ 등 관리자 판단하에 이상할 시 밴처리 됩니다.
